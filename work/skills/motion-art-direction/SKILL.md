# Motion Art Direction

Convert a content brief into a production-ready visual plan before writing animation code.

## Workflow

1. Lock message budget.
- Keep one headline and one support line per slide.
- Keep each visible text block to 12 words or fewer.

2. Build five styleframes first.
- Produce one still per slide.
- Approve hierarchy, spacing, and color before timing.

3. Define system tokens.
- Use spacing, typography, depth, and motion tokens from `references/tokens.md`.
- Reuse tokens across all slides for consistency.

4. Map composition rhythm.
- Alternate left, right, and center focal blocks.
- Avoid repeating the same card geometry for consecutive slides.

5. Author animation in passes.
- Pass 1: static layout only.
- Pass 2: enter and exit motion.
- Pass 3: decorative motion and texture.
- Pass 4: quality gate and final render.

## Composition Rules

1. Place the focal card in top, middle, and lower thirds across the sequence to create vertical rhythm.
2. Keep a persistent shared motif across slides (progress tracker, accent rail, or badge family).
3. Reserve one clear negative-space zone per frame to avoid visual saturation.
4. Keep depth levels explicit: background atmosphere, content plane, foreground accents.
5. Treat transitions as semantic punctuation, not default fades.

## Typography Rules

1. Use one display size for headlines and one body size for support lines.
2. Keep support text at stronger contrast than decorative labels.
3. Keep line length short for mobile-first vertical videos.
4. Avoid paragraph blocks in instructional infographic slides.
