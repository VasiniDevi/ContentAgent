# Default Thresholds

## Text Budget

- Maximum words per visible text block: 12

## Render Metadata

- Width: 1080
- Height: 1920
- FPS: 30
- Duration range: 33.0 to 37.0 seconds for 5-slide infographics

## Contrast

- Body and headline pairs: minimum 4.5:1
- Decorative label pairs: minimum 3.0:1

## File Sanity

- MP4 must exist and be non-empty
- Frame count must be greater than 0
